Ouvrez ce fichier dans Chrome sur les deux appareils.

⛔ Microphonie bloquée : Chrome exige HTTPS pour accéder au micro.
Ce fichier est ouvert en file:// — cela ne fonctionne pas sur mobile.

Solution rapide : hébergez ce fichier sur un serveur HTTPS :
• app.netlify.com/drop (glisser-déposer, gratuit)
• Ou sur votre PC : npx serve .
