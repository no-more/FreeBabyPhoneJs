# FreeBabyPhoneJs

WebRTC-based baby monitor PWA. Two phones, same Wi-Fi, QR-code pairing, no server.

> **Rebuild in progress.** The project is being rewritten with **Ionic + Angular** (standalone components, signals, Angular 18+). The previous vanilla TypeScript implementation has been archived under [`legacy/`](./legacy/) and is no longer deployed.
>
> The new app lives at the repo root and will be progressively filled in — see commit history.

## Stack (new)

- **Angular** (standalone components, signals, control flow)
- **Ionic 8** — mobile-first UI components
- **TypeScript strict**
- **PWA** via `@angular/service-worker`
- **Capacitor** — not included initially, may be added later for native packaging
